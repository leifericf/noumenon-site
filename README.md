# Noumenon Site

Source for [noumenon.leifericf.com](https://noumenon.leifericf.com). Static site generated by [Stasis](https://github.com/magnars/stasis), with [Hiccup](https://github.com/weavejester/hiccup) for HTML and [Garden](https://github.com/noprompt/garden) for CSS.

## Layout

- `src/noumenon_site/build.clj` — Stasis page map and `build-site!` entry point
- `src/noumenon_site/render.clj` — shared page chrome (head, nav, footer)
- `src/noumenon_site/styles.clj` — Garden CSS (inlined into each page)
- `src/noumenon_site/content/` — one namespace per page; each returns Hiccup
- `resources/public/` — static assets (favicon, install script, CNAME) copied verbatim into `_site/`
- `dev/user.clj` — Ring/Jetty preview server
- `_site/` — build output (gitignored)

The OpenAPI spec is **not** committed here — `build.clj` fetches the canonical version from [`leifericf/noumenon/resources/openapi.yaml`](https://github.com/leifericf/noumenon/blob/main/resources/openapi.yaml) at build time. The deploy workflow runs on every push to `main` and on a daily cron, so the published `/openapi.yaml` stays fresh without a separate sync job.

## Develop

```bash
clj -X:build              # build _site/ once
clj -M:dev                # REPL
```

In the REPL:

```clojure
(start!)                  ; serves http://localhost:3000
(stop!)
```

Edit content in `src/noumenon_site/content/` and reload — the dev server reads the page map fresh on each request.

## Deploy

Pushes to `main` (and a daily cron) trigger `.github/workflows/deploy-pages.yml`, which runs `clj -X:build` and deploys `_site/` to GitHub Pages.
