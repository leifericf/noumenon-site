Self-improving introspection

An autonomous loop that proposes changes, evaluates them against the benchmark, and keeps what works. Inspired by autoresearch.

Closed-loop optimization

Propose a change, re-evaluate, keep or revert. Five targets: prompts, examples, rules, source code, and model config.

Budget-controlled

Cap by iterations, wall-clock hours, or dollar cost. Run --max-hours 8 for an overnight session with automatic stopping.

On-device ML model

A pure-Clojure neural network learns which Datalog queries to try first. Trains locally with zero token cost.

Features

Git history as a first-class citizen

Commits, authorship, change frequency, co-changed files, and fix-commit ratios are all queryable.

MCP server

Expose the full knowledge graph as MCP tools for Claude Desktop, Claude Code, or any MCP client.

Multi-language imports

Deterministic import graphs for Clojure, Python, JS/TS, Rust, Java, C#, C/C++, Go, Elixir, and Erlang.

Cost transparency

Token estimates before you start, per-file telemetry as it runs, and aggregate totals when done.

Sensitive file protection

.env, *.pem, credentials — tracked as entities but never read or sent to LLMs.

Concurrent & incremental

Up to 20 parallel workers with automatic retry. Incremental sync picks up new commits only.

Named queries

54 pre-built Datalog queries in EDN. Deterministic, reproducible, composable. Run from the CLI or MCP server.

  • Complexity and change hotspots
  • Bug-prone files and co-change pairs
  • Contributor and dependency graphs
  • LLM cost tracking by model and file

Get started

One command to install, zero dependencies. See the README for details.

1

Install

Single binary, auto-downloads everything it needs on first use.

curl -sSL https://noumenon.leifericf.com/install | bash
2

Import a repository

Local path or remote URL. Import is deterministic and takes seconds for most repos.

noum import /path/to/your/repo
3

Query or ask

Named queries for instant results, or natural-language questions with iterative LLM reasoning.

noum ask /path/to/your/repo "Which files have the most risk?"
